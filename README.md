# Descripción del proyecto

Este proyecto corresponde al desarrollo del parcial sobre **estructuras de datos no lineales**, en el cual se implementan diferentes estructuras utilizando el lenguaje de programación Python.

El objetivo principal del trabajo es comprender cómo funcionan estructuras como árboles, tries, tablas hash, heaps y colas con prioridad, así como analizar su funcionamiento, sus operaciones principales y sus aplicaciones en problemas reales.

A lo largo del proyecto se desarrollan **15 ejercicios**, en los cuales se implementan diferentes estructuras de datos utilizando programación orientada a objetos. Cada ejercicio incluye su respectiva clase, documentación del código mediante docstrings y ejemplos de uso para verificar su funcionamiento.

---

# Estructuras de datos implementadas

En este proyecto se implementaron las siguientes estructuras:

* Árboles N-arios
* Tries
* Tablas Hash
* Heaps
* Colas con prioridad
* Árboles de expresión
* Árbol de Huffman

Estas estructuras son utilizadas en muchos sistemas reales como motores de búsqueda, correctores ortográficos, compresión de datos y sistemas de planificación de tareas.

# Cómo ejecutar el código

Para ejecutar cualquiera de los ejercicios se deben seguir los siguientes pasos:

1. Abrir el archivo del ejercicio en Python.
2. Ejecutar el programa.
3. Observar el ejemplo incluido al final de cada ejercicio, el cual demuestra el funcionamiento de la estructura implementada.

Ejemplo de ejecución desde la terminal:

```
python ejercicio11.py
```

Cada archivo contiene un ejemplo que muestra cómo insertar datos y realizar operaciones con la estructura de datos correspondiente.
